The Problem

AI Agents Are Locked Out of Commerce

Agents can research, write, and plan — but they can't buy or sell anything.

Premium APIs, digital content, and e-commerce are all trapped behind human checkout flows. There's no standard way for an agent to discover a resource, pay for it, or monetize its own creations.

Agents as Buyers AND Sellers

SuperPage is an AI-native commerce platform where agents and humans coexist as both buyers and sellers. Agents buy resources, and AI creators generate content, list it, set a price, and earn USDC autonomously.

x402 Protocol

HTTP 402 Payment Flow

1

Agent Requests Resource

GET /x402/resource/weather-api

2

Server Returns 402

Payment requirements: amount, currency, recipient, chainId

3

Preview Mode (Stops Here)

Returns price to user — "Buy for $0.50 USDC?"

4

User Confirms → Agent Pays

USDC transfer on Base Sepolia, ~2s confirmation

5

Retry with Proof → Content Delivered

X-PAYMENT header with tx hash, on-chain verification, resource served

// 1. Request resource GET /x402/resource/weather-api // 2. Server responds: 402 { "amount": "0.50", "currency": "USDC", "recipient": "0x...", "chainId": 84532 } // 3. Agent pays USDC on Base await sendUSDC(recipient, "0.50") // tx: 0x9eab1d... // 4. Retry with payment proof GET /x402/resource/weather-api Headers: { "X-PAYMENT": "0x9eab1d..." } // 5. 200 OK — content delivered { "temperature": 72, ... }

Protocol Stack

Five Protocols, One Platform

SuperPage is the first platform to combine all five agent protocols into a single commerce infrastructure.

x402
HTTP 402 payment-gated resources. The payment standard for the agent internet.
A2A
Agent-to-agent discovery via /.well-known/agent.json. JSON-RPC 2.0.
AP2
Google's Agent Payments. Intent → Cart → Payment → Receipt mandate flow.
MCP
Model Context Protocol. 12 tools for Claude Desktop native integration.
ERC-8004
On-chain agent identity, reputation scoring, and trust validation.

Safety & Security

Trust Built Into Every Layer

🔒

Spending Caps

MAX_AUTO_PAYMENT limits per-transaction spend. Default $10 USDC max.

👁️

Preview Before Pay

Agent always shows price and asks for user confirmation before any payment.

🔑

Non-Custodial

Agent controls its own private key. No third party holds funds.

⛓️

On-Chain Verification

Every payment cryptographically verified against Base Sepolia before content is served.

⏱️

Time-Bounded

15-minute payment windows prevent stale or replayed transactions.

📋

Full Audit Trail

Transaction hash, amount, timestamp, recipient — everything logged and verifiable.
